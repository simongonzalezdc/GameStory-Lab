# Creator Production Suite

`GameStory-Lab` is the canonical repo for the Creator Production Suite.

Imported modules preserved with git history:

- `modules/shiplab/` from `Pastorsimon1798/ShipLab`
- `modules/generative-assets-lab/` from `Pastorsimon1798/Generative-Assets-Lab`
- `modules/generative-score-lab/` from `Pastorsimon1798/Generative-Score-Lab`

## Suite map

1. **GameStory-Lab** — game concept, mechanics, lore, validation, documentation, pitch/GDD export.
2. **Generative-Assets-Lab** — local-first 2D game asset generation and refinement.
3. **Generative-Score-Lab** — adaptive game soundtrack / MIDI / scene-based composition.
4. **ShipLab** — post-production packaging, docs, licensing, marketing, and deployment guidance.

## Boundary rule

Modules are preserved under `modules/` first. Do not mix code directly into the root workspaces until each integration has a written spec, dependency plan, and tests.

## Not part of this suite

`EvoLab` is explicitly not part of this suite.
