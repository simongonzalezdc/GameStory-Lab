# Frontend Visual Polish Plan

## 1. Chat Spinner Redesign (Garnet + Amethyst)

- **Objective**: Create a distinct, magical loading indicator that differentiates the AI processing state.
- **Implementation**:
- Update `packages/frontend/src/index.css` to define a new `--gradient-garnet-amethyst`.
- Refactor `.jewel-spinner` to use this gradient.
- **Enhancement**: Add a second orbiting layer (using `::after` or a child element) rotating in the opposite direction for a "gyroscope" or "magical rune" effect.
- **Animation**: Tune `jewel-spin` to be smoother and `jewel-pulse` to be more "breathing" than "blinking".

## 2. Visual Gloss & Eye Candy

- **Shimmering Skeletons**:
- Create a `.skeleton-shimmer` class in `index.css` replacing standard gray pulses with a subtle moving gradient (Garnet/Amethyst/Surface).
- Apply to `ProjectsPage` loading state and `ProjectAssistantPanel` thinking state.
- **Ambient Background Elements**:
- Add a "floating orb" animation (`@keyframes float`) to the `CursorGlow` or a new background component to give life to static pages.
- **Interactive Borders**:
- Add a `group-hover` effect to `SpotlightCard` that gently pulses the border color using the Amethyst tint.

## 3. "More Awesome" Spinner Features

- **Morphing Geometry**: Animate the `border-radius` of the spinner from circle -> squircle -> diamond -> circle to simulate "shape-shifting" magic.
- **Status Text**: Create a `ThinkingIndicator` component (to replace simple text) that fades through messages like "Analyzing...", "Crafting...", "Polishing..." while the spinner is active.
- **Particle Trails**: Add tiny CSS-only particles (using `box-shadow` cloning) that trail behind the main spinner rotation.

## 4. Execution Steps

1.  Define new gradients/animations in `index.css`.
2.  Update `.jewel-spinner` CSS.
3.  Create `ThinkingIndicator` component in `packages/frontend/src/components/ThinkingIndicator.tsx`.
4.  Integrate into `ProjectAssistantPanel`.
5.  Update loading skeletons in `ProjectsPage`.
